Testing for Data Scientists

How data scientists can incorporate testing into their development, delivered at PyData Ann Arbor in Jan 2020.

In this talk, I use two examples to illustrate how testing can be better incorporated into a data scientist's workflow. At the same time, I also introduce two libraries, Hypothesis and Great Expectations, which can help with testing code and data.
